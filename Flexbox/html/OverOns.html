Over Ons

Wat is SecureSend?

SecureSend is een website die ervoor zorgt dat men gegevens veilig kan doorsturen. Het is opgericht omdat er weinig veilige websites zijn waar je gegevens veilig kunt doorsturen.

Is SecureSend veilig?

Ja, onze code is opensource dus je kan alles terugvinden. We kunnen zelf niet aan de opgeslagen gegevens omdat we de key niet hebben om het te kunnen ontgrendelen.

Hoe werkt SecureSend?

Wanneer men een bericht/wachtwoord in getypt heeft en op de creër link knop heeft geduwd. Word het bericht geëncryteerd. Niemand kan aan deze gegevens komen enkel als ze de sleutel ervoor hebben. De sleutel word niet opgeslaan in een databank maar meegegeven in de link. Wanneer men op de link klikt word het bericht gedecrypteerd waardoor het bericht zichtbaar wordt voor de ontvanger. Tegelijkertijd word het bericht verwijdert van de database.

Wat is SecureSend's doel?

Ons doel is om volwassenen een manier te geven om berichten/wachtwoorden/gegevens wereldwijd veilig met elkaar te delen.
